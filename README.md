# MoonLaunch

Flutter mobile app and Laravel backend for trading PancakeSwap token pairs — monitors new launches and executes buy/sell.

## Features
- Real-time new pair detection on PancakeSwap
- One-tap buy/sell execution
- Portfolio tracking
- Push notifications for price alerts

## Tech Stack
- Flutter (Dart) — mobile frontend
- Laravel — API backend
- BSC / PancakeSwap SDK

## Build
```bash
flutter pub get
flutter run
```
